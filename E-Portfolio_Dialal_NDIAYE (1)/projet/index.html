Projets Professionnels

Migration de Dashboard

J’ai participé à la migration de dashboards initialement basés sur des Data Modules vers un framework unifié, afin d’établir une source de données unique pour tous les rapports. Cette migration a permis d’améliorer la cohérence et la performance des analyses. J’ai utilisé SQL et SCI (outil de reporting basé sur IBM Cognos) pour assurer la transition et optimiser l’accès aux données.

Automatisation avec Python

J’ai développé un script Python permettant d’identifier automatiquement les modifications apportées à un fichier Excel. Auparavant, les collaborateurs recevaient un fichier modifié par leur responsable sans indication des changements effectués. Mon script compare les versions, détecte les lignes modifiées et génère un fichier Excel final listant les différences (numéro de ligne et contenu modifié), optimisant ainsi le suivi des corrections et réduisant le temps d’analyse.
